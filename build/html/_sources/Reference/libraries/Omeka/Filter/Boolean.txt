--------------------
Omeka_Filter_Boolean
--------------------

.. php:class:: Omeka_Filter_Boolean

    Package: Filter

    A Zend_Filter implementation that converts any boolean value passed to it to 
    an integer: 1 or 0.

    .. php:method:: filter($value)
    
        Filter the value
        
        :param unknown $value: 
        :returns: string "1" or "0"
