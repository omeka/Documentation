.. php:function:: lightgallery($files)

    Display lightGallery media viewer.
    
    Only filetypes supported by the lightGallery will be included; use
    lightgallery_other_files to list links to other files not supported in the
    gallery.
    
    :type $files: array
    :param $files: Array of files to display
    :returns: string HTML
