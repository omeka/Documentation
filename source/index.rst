#####################################
Omeka Classic Developer Documentation
#####################################

This is the developer documentation for `Omeka Classic <http://omeka.org/classic>`_. If you're working on an Omeka theme or
plugin, or on Omeka itself, you've come to the right place.

For documentation for end-users and site administrators, or for information about older Omeka versions, please visit the
`Omeka Classic User Manual <http://omeka.org/classic/docs/>`_ instead. For information about Omeka S, please visit the `Omeka S section of omeka.org <http://omeka.org/s>`_.
